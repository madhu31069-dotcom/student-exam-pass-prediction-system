# Student Exam Pass Prediction System

## Description

This project predicts whether a student will pass or fail an examination using academic and behavioral factors such as study hours, attendance rate, past exam scores, and other student-related data. The system uses Logistic Regression and data visualization techniques to analyze student performance and identify students at risk of failing.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly
* Scikit-learn
* KaggleHub

## Features

* Student performance analysis
* Pass/Fail prediction
* Risk category classification
* Exploratory Data Analysis (EDA)
* Interactive Plotly dashboards
* Logistic Regression model
* Correlation analysis

## Dataset

The dataset contains student-related information such as study hours, attendance rate, past exam scores, gender, and final exam scores.

## Model

**Logistic Regression** is used to classify students as Pass or Fail based on academic performance indicators.

## Key Insights

* Students with higher study hours generally achieve better exam scores.
* Attendance rate significantly impacts student performance.
* Past academic performance influences final exam results.
* Risk categories help identify students who need additional academic support.

## Output

* Pass/Fail prediction results
* Risk category analysis
* Statistical summaries
* Visualizations and dashboards
* Confusion Matrix
* Accuracy, Precision, Recall, F1-Score, and ROC-AUC evaluation metrics
